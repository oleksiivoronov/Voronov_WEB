Атрибут < value >

Атрибут value встановлює номер, з якого буде починатися список. value застосовується тільки для нумерованих списків, тоді як тег<li> знаходиться всередині котейнера <ol>.

При цьому не має значення, який тип списка встановлено з допомогою type, атрибут value так само працює и з римськими і з арабськими числами .

Синтаксис

< li value="чило">...< /li>

Специфікація

HTML: 3.2 4.01 5.0 XHTML: 1.0 1.1

Значення

Значення: Значення за замовченням:
Будь-яке додатнє число 1
