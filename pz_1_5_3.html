Тег < li >

Тег < lo > визначає пункти списка і може включати в себе любі інші HTML-елементи (списки, абзаци, зображення).

За замовченням браузери відображають пункти списка із невеликим відступом з лівої сторони.

Синтаксис

< ol >

< li > елемент нумерованого списку < li >

< li > елемент нумерованого списку < li >

< li > елемент нумерованого списку < li >

< /ol >

< ul >

< li > елемент нумерованого списку < li >

< li > елемент нумерованого списку < li >

< li > елемент нумерованого списку < li >

< /ul >

Атрибути

Атрибут Призначення
class Визначає імя класу для елементу
dir Визначає напрям тексту для контента в елементі
id Визначає унікальний id для елемента
lang Визначає код мови для контента в елементі
title Визначає додаткову інформацію про елемнт
style Визначає інлайновий стиль для елемнту
