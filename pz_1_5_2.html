МІЙ ЕЛЕКТРОННИЙ ДОВІДНИК HTML
Призначення Назва тегу Короткий опис Стандарт
Для каркасу документа < html > Вказує початок і кінець HTML документа HTML4
< head > містить інформацію, що сточується документа в цілому HTML5
< body > Охоплює ту частину HTML-документа (текст, зображення й елементи формування), що буде видна користувачеві HTML5
< title > Тег TITLE установлює заголовок HTML-документа, виведений у рядку заголовка вікна броузера HTML5
< body alink=? > Атрибут ALINK установлює кольори всім посиланням HTML-документа, на якіпоміщений курсор миші, але до того моменту, як кнопка миші натиснута. Колір може бути зазначений за допомогою назви або шістнадцяткового коду HTML5
Для роботи з текстом < font size=? > Атрибут SIZE тегу FONT установлює відносний розмір шрифту. Список можливих значень складається з додатних і від’ємних чисел від 0 до 7 HTML4
< font color=? > Атрибут COLOR тегу FONT установлює кольори тексту. Колір може бути зазначений за допомогою за допомогою назви або шістнадцяткового коду HTML4
< p align=? > Атрибут ALIGN тегу P установлює режим вирівнювання тексту: уліво (LEFT), вправо (RIGHT) або по центрі (CENTER) HTML4
< ol > Тег OL («ordered list» – упорядкований список) установлює режим послідовної нумерації елементів, розміщених усередині списку HTML5
< ul > Тег UL («unordered list» – неупорядкований список) установлює режим відображення тексту у вигляді маркірованого списку HTML5
Для гіперпосилань < a href=»URL» > Тег A встановлює зв’язок з деякою точкою усередині того ж HTML-документа або з іншим URL (гипертекстове посилання). Атрибут HREF тегу A описує об’єкт, що представляє собою текст або малюнок усередині HTML-документа, або текст або малюнок у зовнішньому документі HTML4
Для таблиць < table > Тег TABLE установлює початок і кінець таблиці. Всі теги, що визначають структуру таблиці, повинні розташовуватися між тегами TABLE HTML4
< tr > Тег TR визначає рядок таблиці HTML5
< td > Тег TD визначає колонки таблиці HTML5
< th > Тег TH установлює режим відображення тексту у вигляді заголовка таблиці HTML4
< table border=? > Атрибут BORDER тегу TABLE установлює ширину рамки таблиці в пікселях HTML5
< table cellspacing=? > Атрибут CELLSPACING тегу TABLE установлює відстань між сусідніми комірками таблиці в пікселях HTML5
< table width=? > Атрибут WIDTH тегу TABLE установлює ширину таблиці в пікселях або у відсотках від ширини області перегляду HTML4
